The Dynamic Showcase Carousel is a plugin that allows you to easily create a product carousel, ideal for e-commerce websites looking to display products or promotions in an eye-catching format.

== Description ==
The Dynamic Showcase Carousel is a plugin that allows you to easily create a product carousel, ideal for e-commerce websites looking to display products or promotions in an eye-catching format.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress 'Plugins' screen directly.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the plugin's settings page to customize the carousel as desired.

== Frequently Asked Questions ==
= Can I customize the carousel's design? =
Yes, you can customize the design of the carousel from the plugin's settings page.

= Is this plugin mobile-responsive? =
Yes, this plugin is designed to be responsive and works on both mobile devices and tablets.
